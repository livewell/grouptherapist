# GroupMe NodeJS Callback Bot 

## Overview

GroupMe bot written in NodeJS, capable of responding to basic function calls ie. /roll, /weather, /food etc.

## Commands

  * /roll ; return random 1-100 integer
  * /coinflip ; return heads or tails
